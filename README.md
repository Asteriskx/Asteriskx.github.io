# Asteriskx.github.io

Portfolio site — [asteriskx.net](https://asteriskx.net)

Built with React Router v7 (SSG) + Three.js particle background, deployed to GitHub Pages.

---

## Tech Stack

| Category | Library / Tool |
|---|---|
| Framework | [React Router v7](https://reactrouter.com/) (React 19, SSG mode) |
| 3D / Particles | [Three.js](https://threejs.org/) |
| Blog | [MDX](https://mdxjs.com/) + rehype-highlight |
| Build | [Vite](https://vitejs.dev/) |
| Language | TypeScript |
| Font | Zen Kaku Gothic New (Google Fonts) |
